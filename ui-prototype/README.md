# 🎨 Stage 1: Static Frontend UI Prototype

## 📌 What is this?
This folder contains the **High-Fidelity Frontend Prototype** for the EasyReal Dashboard. It is a "design-first" implementation built using native HTML, CSS, and JavaScript. 

The purpose of this stage is to finalize the user interface (UI) and user experience (UX) for a non-tech-savvy user (Nanay) before introducing complex backend databases.

## ⚙️ Prototype Features
* **Zero Dependencies:** Runs instantly in any browser by opening `index.html`.
* **State-Driven UI:** Uses a centralized JavaScript render engine that mimics how modern frameworks handle data.
* **Database-Ready Architecture:** Data is isolated in a `mockDatabase` object at the top of the script file, making it ready to be swapped with API `fetch()` requests in Stage 2.
* **Responsive Layout:** Automatically scales down seamlessly for mobile phone screens.

## 🚀 How to View Locally
1. Clone or download this repository.
2. Navigate into the `ui-prototype` folder.
3. Double-click `index.html` to open and interact with the dashboard in your web browser.
